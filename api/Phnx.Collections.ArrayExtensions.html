BinarySearch<T>(T[], T)

Searches a range of elements in a sorted array for a value

Declaration
public static int BinarySearch<T>(this T[] source, T value)
Parameters
Type Name Description
T[] source

The array to search.
T value

The object to search for
Returns
Type Description
System.Int32

The index of the specified value in the specified source, if value is found; otherwise, a negative number. If value is not found and value is less than one or more elements in source, the negative number returned is the bitwise complement of the index of the first element that is larger than value. If value is not found and value is greater than all elements in source, the negative number returned is the bitwise complement of the index of the last element plus 1. If this method is called with a non-sorted source, the return value can be incorrect and a negative number could be returned, even if value is present in source.
