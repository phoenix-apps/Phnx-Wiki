Class CrudDbSet<TDbContext, TEntity, TKey>

Provides Create, Read, Update and Delete operations on data, via Microsoft.EntityFrameworkCore. This does not save changes, so you'll need to use Microsoft.EntityFrameworkCore.DbContext.SaveChanges whenever changing data
